How access works

For each presentation you can choose either: Public (open link) or Protected (OneDrive/SharePoint link with provider-managed password or restricted recipients).

Security is enforced by the storage provider link settings, not by browser JavaScript.

How to add a new presentation

  1. Duplicate one card in this page.
  2. Set title, date/source, and add a short abstract below the title.
  3. Choose badge: badge-public or badge-protected.
  4. For protected slides, use a provider-managed protected link (OneDrive/SharePoint).
